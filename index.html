Assignment - Open Source Cloud Platform

Introduction

This is the assignment submission for Software Lab 3, focusing on the topic of "Open Source Cloud Platform". In this project, we explore various open-source cloud technologies and their applications. The project repository is available at: GitHub Repository.

Objective

The primary goal of this assignment is to analyze and understand the real-life applications and benefits of open-source cloud platforms. We had on the following tasks:

  • 1. Integration of Gmail with Google Calendar
  • 2. Create a Google form for conducting survey
  • 3. Create Google Charts based on the response and embed into onto the Website
  • 4. Explore Microsoft Azure
  • 5. Accessing Google maps via APIs
